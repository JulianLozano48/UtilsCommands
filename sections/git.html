Started

  • Clonar repository: git clone 'url'

    • Branchs

  • Listar ramas: git branch
  • Crear nueva rama: git branch 'branch_name'
  • Cambiar rama: git checkout 'branch_name'
  • Crear nueva rama en base a la actual: git checkout -b 'branch_name'
  • Fusiona rama: git merge 'branch_name'

    • Commits

  • Crear commit con mensaje: git commit -m "message"
  • Mostrar estado: git status
  • Agregar archivo: git add 'name_file'
  • Agregar todos los archivo: git add .

    • Pushs

  • Subir cambios: git push
  • Subir cambios SIN rama publicada: git push --set-upstream origin 'branch'

    • Pulls

  • Bajar cambios de una rama remota: git pull origin 'branch' --no-rebase

    • Revertir y corregir

  • Remover archivo que se ha añadido: git reset 'name_-file'
  • Revierte todo al ultimo commit: git reset --hard
  • Revierte un archivo a su última versión: git checkout -- 'name_-file'
  • Crea un nuevo commit que revierte uno anterior: git revert 'id_commit'
